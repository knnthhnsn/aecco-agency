# Indholdsplan - æccoo

## Formål
At uddanne og informere kunden om værdien af bæredygtigt webdesign, samtidig med at vi viser vores tekniske og visuelle kompetencer.

## Kerneindhold
- **Hero**: Værdisæt og "Sustainable Futurism" visuel identitet.
- **Services**: Beskrivelse af vores arbejdsmetoder.
- **Uddannelse**: Hvad er bæredygtigt webdesign? (Statistik på CO2-udledning).
- **Case Studies / Metoder**: Low-energy design, SVG-optimering, hurtig kode.

## Formidlingsformater
- **Tekst**: Argumentation for bæredygtighed.
- **Visuelt**: Motion Graphics (CSS-baseret) for at demonstrere performance.
